### 用法
- 运行应用: `python dev.py run`
- 自动重载: `python dev.py watch`（先安装 `cargo install cargo-watch`）
- 检查/格式化/静态检查/测试:
  - `python dev.py check`
  - `python dev.py fmt`
  - `python dev.py clippy`
  - `python dev.py test`
- 数据库（SQLite）:
  - 迁移: `python dev.py migrate`（执行 `migrations/*.sql`）
  - 重置: `python dev.py reset-db`（删除 DB 并重建）
  - 种子数据: `python dev.py seed`

- 脚本会自动从 `Rocket.toml` 读取 `default.databases.sqlite_db.url`（例如 `sqlite:./database/todos.db`），无需手动配置路径。
